    Configuración Interactiva: Rawncher y la Gestión de Trackers

    En la RaW Suite, creemos que tu tiempo debe dedicarse a la preservación del contenido, no a pelearte con archivos de configuración complejos. Por eso, RawLoadrr incluye Rawncher, un asistente interactivo (rawncher.py) que automatiza la configuración de trackers y la gestión de sesiones.

    🚀 El Menú Principal de Rawncher

    Al ejecutar python3 rawncher.py (o pulsar 2 en el menú de Singularity), accederás a la torre de control:

    1. Usar tracker existente: Lanza subidas rápidas usando perfiles ya configurados.
    2. Configurar tracker existente: Edita claves API, URLs de announce o preferencias de anonimato.
    3. Crear nuevo tracker: La opción estrella. Configura un nuevo sitio UNIT3D desde cero de forma asistida.
    4. Modo debug: Prueba el pipeline sin riesgo de baneo o subidas erróneas.
    5. Listar/Cargar sesiones: Recupera trabajos interrumpidos o revisa el historial.

    🏗️ Creación Automática de Trackers (Opción [3])

    ¿Has entrado en un nuevo tracker UNIT3D? No necesitas editar código Python. El asistente te pedirá:

    • Nombre/Abreviación: (ej: NUEVOTRK).
    • URL Base: La dirección web del tracker.
    • API Key: Tu token de acceso personal.
    • Passkey/Announce: Tu URL de announce para generar los torrents.

    ¿Qué sucede por detrás? Rawncher genera automáticamente un módulo .py en src/trackers/ con toda la lógica necesaria y actualiza tu data/config.py. El nuevo tracker estará listo para recibir inyecciones de inmediato.

    🔧 Configuración Detallada (Opción [2])

    Si necesitas ajustar un tracker que ya funciona, este menú te permite:

    • Cambiar entre Modo API y Modo HTTP (para trackers con protecciones especiales).
    • Activar/Desactivar el Anonimato por defecto.
    • Configurar el Draft Mode (borradores), ideal para trackers como BHD donde prefieres revisar la subida antes de publicarla.
    • Ajustar las Firmas BBCode personalizadas por sitio.

    📂 Flujos de Trabajo Inteligentes

    Al elegir un tracker para subir (Opción [1]), Rawncher te ofrecerá tres caminos:

    1. Subir desde una ruta (Fácil)

    Escribes la ruta de la carpeta o el archivo. Rawncher detectará si es un MKV, una serie o una película y configurará los argumentos base por ti.

    2. Usar lista existente (.txt)

    Si ya tienes una lista de rutas (por ejemplo, generada por un script externo), RawLoadrr las procesará una por una en modo batch.

    3. Triage primero (Recomendado)

    Lanza el motor de Triage MKV. Escaneará el directorio que le indiques y separará el contenido en listas: - HEVC: Todo lo que sea x265 nativo. - H264: Contenido legacy o de alta compatibilidad. - Packs: Temporadas o colecciones.

    Una vez terminado el Triage, podrás elegir cuál de estas listas quieres empezar a subir al tracker seleccionado.

    Consejo de Experto: Usa siempre el Modo Triage para subidas masivas. Te permite priorizar el contenido HEVC, que suele tener mejor acogida y consume menos espacio en el servidor del tracker.
