    Auditoría de Binarios (Technical Audit)

    MKVerything usa herramientas muy potentes que funcionan "por debajo". Aquí tienes los comandos reales que lanzan y qué hacen explicados de forma muy fácil.

    Herramientas de Análisis

    1. FFprobe

    • Comando real: ffprobe -v quiet -print_format json -show_format -show_streams "archivo.mp4"
    • Explicación para Mortales: Es como una máquina de rayos X. Mira el video sin abrirlo y nos dice cuánto dura, si tiene sonido y si es grande o pequeño. Todo lo escribe en un papel (JSON) para que el ordenador lo entienda.

    2. MediaInfo

    • Comando real: mediainfo --Output=JSON "archivo.mkv"
    • Explicación para Mortales: Es el ayudante del médico. Si los rayos X (FFprobe) no funcionan porque el video está muy roto, MediaInfo intenta leer las etiquetas de la caja para decirnos qué hay dentro.

    3. mkvmerge (Identificación)

    • Comando real: mkvmerge -J "archivo.mkv"
    • Explicación para Mortales: Es como un inspector de piezas de LEGO. Mira cómo están encajadas todas las piezas del video y nos dice si alguna está suelta o falta.

    Herramientas de Procesamiento

    4. FFmpeg (Prueba de integridad)

    • Comando real: ffmpeg -v error -xerror -i "archivo.mp4" -f null -
    • Explicación para Mortales: Es la prueba de fuego. El ordenador intenta leer el video entero de principio a fin súper rápido. Si encuentra un solo trocito roto, se para y nos avisa: "¡Este video está mal!".

    5. FFmpeg (Rescate/Conversión)

    • Comando real: ffmpeg -y -i "viejo.avi" -c:v libx264 -crf 23 -c:a aac "nuevo.mkv"
    • Explicación para Mortales: Es la máquina de reciclaje. Coge un video viejo y oxidado (como un .avi) y lo convierte en un video moderno y brillante (.mkv) que funciona en todas partes.

    6. MakeMKV

    • Comando real: makemkvcon mkv iso:"pelicula.iso" all "carpeta_destino"
    • Explicación para Mortales: Es el extractor mágico. Abre los archivos .iso (que son como cajas cerradas con llave) y saca todas las películas que hay dentro para que podamos verlas sin problemas.

    7. VapourSynth (Nativo en Docker)

    • Comando real: vspipe script.vpy - --y4m | ffmpeg -i - -c:v libx264 "salida.mkv"
    • Explicación para Mortales: Es la máquina de efectos especiales. El Cirujano la usa para "pulir" el video antes de reciclarlo, quitando el ruido y las manchas de las películas viejas. Al usar Docker, esta máquina ya viene montada y lista para usarse.

    Nota: Estos comandos son el motor que hace que MKVerything sea tan seguro.
